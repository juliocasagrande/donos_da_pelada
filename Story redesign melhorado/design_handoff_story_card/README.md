# Handoff: Post-Pelada Player Result Story (9:16)

## Overview
A shareable Instagram Story / Reel card generated by the "Donos da Pelada" app after a pickup-football ("pelada") match closes. Shows one player's result: crowd rating, season stats, a quote, and an app-download CTA. Redesigned from a flat, single-tone green version into a layered composition using the app's design system.

Two variants are included, same structure/layout, different theme + copy:
1. **`Story Casagrande.dc.html`** — standard match-closed result card (dark green hero).
2. **`Story Craque da Pelada.dc.html`** — "player of the match" / MVP variant (gold hero, crown icon, "Nx craque" badge). Use this variant whenever the featured player is voted MVP of the pelada.

## About the Design Files
The file in this bundle (`Story Casagrande.dc.html`) is a **design reference built in HTML** — a static prototype at fixed pixel dimensions showing the intended look, exact colors, type, and spacing. It is not production code to embed as-is. The task is to **recreate this layout in the app's existing environment** (whatever renders the app's Stories today — e.g. a native canvas/image-composition pipeline, a web view, React Native, etc.), using that environment's existing patterns for text rendering, image compositing, and asset export. If no such pipeline exists yet, choose the most appropriate approach for generating a shareable 1080×1920 image/video in the target stack.

## Fidelity
**High-fidelity.** All colors, fonts, sizes, spacing, and copy below are final — implement pixel-for-pixel where the target renderer allows.

## Interactions & Behavior
Static share image — no interactivity, animation, or states. Generated server-/client-side per match result and exported as an image (or video frame) for Instagram Stories/Reels.

## State Management
Content is data-driven per player/match:
- Player name, position/team label, photo
- Pelada attendance count (badge)
- Crowd rating (star count + reviewer count) — currently unrated state (0 stars filled, "0 jogadores avaliaram")
- Season stats: goals, assists, presença (appearances)
- Quote (player-submitted or auto-generated)
- Match label + date
